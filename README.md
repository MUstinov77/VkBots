## Запуск проекта:
  1. Заполните файл .env как показано в .env.example
  2. Выполните docker compose up -d
  3. Взаимодействуйте с проектом через http://localhost:8000/docs/
  4. Для всех CRUD операций необходима регистрация и авторизация.
